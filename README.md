🏏 WicketPass
WicketPass is a blockchain-powered PSL ticketing and fan loyalty platform built on WireFluid Network.
It eliminates ticket fraud, enables secure resale, and creates a permanent on-chain identity for fans.
🔗 Live Demo: https://wicketpass.vercel.app
🚀 Features
🎟️ NFT Ticketing
Every ticket is minted as an NFT — impossible to fake or duplicate.
🪪 Fan Passport
On-chain fan identity tracking attendance, loyalty, and reputation.
💸 Resale Marketplace
Secure ticket resale with enforced price caps and royalties.
🏟️ Gate Verification
QR-based ticket validation with real-time blockchain verification.
🛠️ Admin Dashboard
Manage users, tickets, blacklist, and rewards.
🧱 Tech Stack
Blockchain: WireFluid Network
Smart Contracts: Solidity
Frontend: React + Vite
Web3 Integration: ethers.js
Styling: Tailwind CSS

🔑 Key Concepts
On-chain Ownership: Tickets exist in user wallets
Loyalty System: Fans earn tiers based on activity
Transparency: Every action is verifiable on-chain
Security: No fake tickets or unauthorized resale
